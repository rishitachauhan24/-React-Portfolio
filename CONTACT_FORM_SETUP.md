# Contact Form Setup - SUPER EASY! 🚀

## Web3Forms - Sirf 30 Seconds Mein Setup! ✨

### Quick Setup (Bilkul Free!)

1. **Website kholo**: https://web3forms.com/

2. **Access Key Lo:**
   - "Get Started For Free" button click karo
   - Apna email address daalo (rishitachauhan24@navgurukul.org)
   - "Get Access Key" click karo
   - Turant email mein access key aa jayega! 📧

3. **Code Mein Daalo:**
   - `src/pages/Contact.jsx` file kholo
   - Line 18 par `access_key` wali line dhundo:
   ```javascript
   access_key: 'ab6036d7-0a69-4327-9147-d7c8e1ac0e68', // REPLACE WITH YOUR KEY
   ```
   - Apni access key se replace karo:
   ```javascript
   access_key: 'YOUR_ACCESS_KEY_HERE',
   ```

4. **Done! Test Karo! 🎉**
   - Contact form fill karo
   - "Send Message" click karo
   - Email check karo - message aa gaya hoga!

### Kaise Kaam Karta Hai?

- **Free**: 250 emails per month absolutely free
- **Instant**: Koi verification nahi, turant kaam karta hai
- **Simple**: Sirf email daalo, key lo, code mein paste karo
- **Reliable**: Messages seedha tumhare email pe aate hain

### Try Karne Ke Liye

Abhi bhi form test kar sakte ho! Demo key already lagayi hui hai. Message send hoga lekin tumhare email pe nahi aayega (demo inbox mein jayega). Apne email pe messages receive karne ke liye apni access key daalo.

## Troubleshooting

**Agar message send nahi ho raha:**
1. Internet connection check karo
2. Browser console (F12) mein errors dekho
3. Access key sahi copy-paste kiya hai ya nahi check karo

**Free Tier Limits:**
- ✅ 250 submissions/month (free)  
- ✅ Unlimited forms
- ✅ Email notifications
- ✅ File uploads (up to 5MB)
